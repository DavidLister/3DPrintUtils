Home_after_every_layer_README.txt

Requires:
	- Python 2.x

Description:
	Adds the gcode to home the x and y axis of a 3D printer after every layer.

	This is intended to be used as a temporary fix for either the X and Y axis losing steps while printing or as an insurance that a skipped step will not ruin a large print.

Usage:
	Can be run via command line or opened through windows explorer.  When run through command prompt, an argumentof the filename is optional, if there are no arguments, there will be a prompt for the file name.
